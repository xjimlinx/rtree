# rtree

> 一个用Rust编写的tree工具的简易版本

使用方法：

```bash
Usage: tree [OPTION] [DIRECTORY]
Options:
  -h, --help               Print this help message
  --depth=DEPTH            Set the maximum depth of the tree (default: 0)
  --all                    Show hidden files and directories
```

编译：

```bash
make build 
```

测试：

```bash
make test
```

安装（将安装到 `/usr/bin 目录下`）：

```bash
make install
```
